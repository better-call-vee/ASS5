Question-1: What are the different ways to select an element in the DOM?
There are several ways to select an element in the DOM. We can use `getElementById` for selecting an element by its ID, `getElementsByClassName` for selecting elements with a certain class, or `querySelector`/`querySelectorAll` for selecting elements using CSS selectors.
Question-2: What is the difference between innerHTML, innerText, and textContent?
`innerHTML` gets or sets the HTML content inside an element, meaning it can include tags. `innerText` gives we the visible text inside an element, excluding HTML tags and styles. `textContent` is similar to `innerText`, but it retrieves all the text, even from hidden elements.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where instead of attaching an event listener to each individual element, we attach it to a parent element. The parent listens for events and determines which child element was triggered, improving performance especially with large numbers of elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is a process where an event starts from the target element and bubbles up to its parent elements. For example, if we click a button inside a div, the click event will first fire on the button and then propagate to the div and so on.
Question-5: How do you create, add, and remove elements using JavaScript?
To create an element, use `document.createElement()`. To add it, use `appendChild` or `insertBefore`. To remove an element, use `removeChild` or `remove`. These methods help we dynamically manipulate the DOM.
